Title: Graphing Calculator

----

Date: 14 December 2017

----

Text:

A program built into the [Newton Works](/terms/newton-works) application that uses specific [stationery](/terms/stationery) allowing the user to perform calculations and display the results in a graph or table. Information formatting using the [calculations](/terms/calculations) stationery can be copied and pasted into other [applications](/terms/application) on a Newton device.
