Title: Button Bar

----

Text:

A thick black line with graphical [buttons](/terms/button) that perform functions related to the current [application](/terms/application). Button bar also refers to the set of [icons](/terms/icon) located at the bottom of the Newton screen which are used to launch applications. On [MessagePad 2000](/terms/messagepad-2000) and [MessagePad 2100](/terms/messagepad-2100) devices, the button bar contains user modifiable [soft buttons](/terms/soft-button), all other Newton devices utilize a permanent set of [hard buttons](/terms/hard-button).
