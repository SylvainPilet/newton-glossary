Title: Application Definition Format

----

Text:

A text file format defined by [Revelar](/terms/revelar) that contains a description on how to display, edit, import, and export the data from an application [soup](/terms/soup) so that the [Revelar Connection Utility](/terms/revelar-connection-utility) can handle it.
