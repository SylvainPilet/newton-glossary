Title: Help Browser

----

Text:

An outline-based help screen interface that uses [help book](help-book) data created by the [Newton Book Maker](/terms/newton-book-maker). Generally used to provide information about a specific [applications](/terms/application). The help browser provides only a subset of the capabilities of the [Newton Book Reader](/terms/newton-book-reader).
