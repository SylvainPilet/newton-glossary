Title: Newton Package Installer

----

Text:

Software developed by [Apple](http://www.apple.com/) that allows the user to install [packages](/terms/package) on a Newton device from a [desktop computer](/terms/desktop-computer). Also referred to as [NPI](/terms/npi).
