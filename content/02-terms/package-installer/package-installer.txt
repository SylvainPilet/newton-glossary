Title: Package Installer

----

Text:

A software program that facilitates the installation of [packages](/terms/package) on a Newton device from a [desktop computer](/terms/desktop-computer) or [laptop](/terms/laptop-computer). Examples of package installers include [Newton Backup Express](/terms/newton-backup-express) and [Newton Connection Utilities](/terms/newton-connection-utilities).
