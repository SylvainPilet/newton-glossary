Title: Graphical User Interface

----

Text:

The use of pictures and icons rather than text to represent the input and output of a software program or operating system. A graphic [user interface](/terms/user-interface) generally includes the use of a windowing system, icons, buttons, and dialog boxes to facilitate communication and user interaction. Also referred to as [GUI](/terms/gui).
