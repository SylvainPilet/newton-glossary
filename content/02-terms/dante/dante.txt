Title: Dante

----

Text:

The code name of the [MessagePad 130](/terms/messagepad-130) device. According to some sources, Dante can also refer to version 2.0 of the [Newton OS](/terms/newton-os). The MessagePad 130 has also been referred to by the code name [Quark](/terms/messagepad-130).
