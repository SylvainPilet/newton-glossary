Title: Voyager

----

Text:

The code name for the [ASIC](/terms/asic) chip set used in the [MessagePad 2000](/terms/messagepad-2000), [MessagePad 2100](/terms/messagepad-2100), and [eMate 300](/terms/emate-300) devices. The Voyager chip was developed by [Cirrus Logic](http://www.cirrus.com/), and the designation [Cirrus](/terms/cirrus) is also sometimes used as a code name.
